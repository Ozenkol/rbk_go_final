# Команды приложения

Этот пакет содержит обработчики команд (Command Handlers) согласно паттерну CQRS.

## Document Handlers
- **CreateDocumentHandler**: Создает новый документ, сохраняет метаданные в БД и возвращает URL для загрузки файла в Minio.
- **UpdateDocumentHandler**: Обновляет метаданные документа и возвращает URL для обновления файла.
- **DeleteDocumentHandler**: Удаляет метаданные из БД и соответствующий объект из Minio.
