# Обработчики HTTP запросов

Этот пакет содержит контроллеры (Handlers) для обработки входящих HTTP запросов с использованием фреймворка Gin.

## DocumentHandler
Управляет API эндпоинтами для документов. Извлекает `userID` и `companyID` из JWT токена и взаимодействует с уровнем приложения для выполнения операций с документами и получения URL для работы с Minio.
